# Инструкция по установке и запуску Ustaz.ai

## Требования

- Node.js версии 18 или выше
- npm или yarn

## Шаги установки

1. **Установите зависимости:**

```bash
npm install
```

2. **Запустите проект в режиме разработки:**

```bash
npm run dev
```

3. **Откройте браузер:**

Приложение будет доступно по адресу: `http://localhost:5173`

## Использование

1. **Вход в систему:**
   - На странице входа введите любой email
   - Нажмите "Кіру как мұғалім"
   - Вы будете перенаправлены на Dashboard

2. **Создание голосового отчета:**
   - На Dashboard нажмите "Голосовой отчет"
   - Нажмите кнопку микрофона для начала записи
   - В demo режиме будет использован предустановленный текст
   - Выберите тип документа
   - Нажмите "PDF жасау"
   - Просмотрите, скачайте или поделитесь PDF

3. **Создание WhatsApp сообщений:**
   - На Dashboard нажмите "Шаблоны WhatsApp"
   - Выберите категорию сообщения
   - Заполните форму
   - Просмотрите предварительный просмотр
   - Скопируйте или отправьте через WhatsApp

## Структура данных

Все данные хранятся в localStorage браузера:
- Информация о пользователе
- Созданные PDF документы
- История сообщений

## Решение проблем

### Ошибка при установке зависимостей

Убедитесь, что у вас установлена последняя версия Node.js:
```bash
node --version
```

Если версия ниже 18, обновите Node.js.

### Проблемы с микрофоном

В demo режиме микрофон не обязателен - приложение использует предустановленные тексты.

### PDF не генерируется

Убедитесь, что браузер поддерживает современные веб-стандарты. Рекомендуется использовать:
- Chrome 90+
- Firefox 88+
- Edge 90+
- Safari 14+

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.

Для предварительного просмотра продакшен сборки:
```bash
npm run preview
```
